date: Fall 2006
location: ECS Building, Room 2120
description: This class will introduce ``Web Programming''; the process of writing interactive applications accessible through the WWW. We will develop in the ``LAMP'' environment: Linux, Apache, MySQL and PHP. Emphasis will be on the integration of these components for a useful application; none will be exhaustively covered. Lectures will provide the overview of various concepts and the class will be centered around group development of a semester-long project. Prerequisites include Web familiarity, programming knowledge and database experience. We will only use very simple databases, so that knowledge can be acquired during the class.
